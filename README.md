# Full Stack Web App

A simple web application with authentication and task management.

## Features
- User Signup & Login (JWT)
- Profile Update & Change Password
- Task CRUD (Create, Read, Update, Delete)
- Responsive UI

## Tech Used
- Frontend: React + TailwindCSS
- Backend: Node.js + Express + MongoDB

## How to Run

### Backend
cd backend
npm install
npm start


### Frontend
cd frontend
npm install
npm run dev
